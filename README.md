# Minimarket Digital Integrado

Plataforma SaaS de gestión comercial, inventario y punto de venta multiempresa.

## Tecnologías

- React + Vite
- Node.js + Express
- Prisma ORM
- PostgreSQL (Neon)
- JWT Authentication
- Docker
- GitHub Actions

## Características

- Multiempresa
- Multi sucursal
- Inventario
- Ventas
- Compras
- Auditoría
- Gestión de usuarios
- Roles y permisos
- Soporte técnico
- Dashboard administrativo
